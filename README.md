# Impostor Game
The game in the folder /partePython_IP, has a popular game here in Spain. The name of the game is Impostor, and consists on two teams one that consists on people that know 
a word, and another team who does not now the word.

Each person has to say a similar word and the hole group needs to eliminate one person each round and the game concludes when there has been the same eliminations as possible 
impostors are in the game. An impostor wins if he is not eliminated at this point.

The code can be used to build an APK with Android using buildozer.
